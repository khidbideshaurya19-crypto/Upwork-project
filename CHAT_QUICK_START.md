# Quick Start Guide - Chat Feature

## 🚀 How to Use the New Chat Feature

### For Companies (Freelancers)

#### 1. Starting a Chat from Project Listings
```
Browse Projects → Click "Chat with Client" button → Opens Messages Page
```
- Every project card now has a blue "Chat with Client" button at the bottom
- Click it to instantly start a conversation with the client who posted the project
- You'll be taken directly to the messages page with the chat open

#### 2. Using the Messages Interface

**Left Sidebar:**
- See all your active conversations
- Green badges show unread message counts
- Click any conversation to open it

**Right Side - Chat:**
- Type your message in the input box at the bottom
- Click 📎 to attach files
- Click 📤 to send

### Message Status Indicators

When you send a message, watch the status change:

1. **🕐** Sending... (pending)
2. **✓** Sent (gray single check)
3. **✓✓** Delivered (gray double check)
4. **✓✓** Read (blue double check) ← Like WhatsApp!

### Editing Messages

1. Hover over your sent message
2. Click the **▼** dropdown button
3. Select **✏️ Edit**
4. Change the text
5. Click **Save**
6. Message now shows **(edited)** label

### Deleting Messages

1. Hover over your sent message
2. Click the **▼** dropdown button
3. Select **🗑️ Delete**
4. Confirm deletion
5. Message disappears for both you and the recipient

### Chat Options Menu

Click **⋮** (three dots) in the chat header for:
- **Clear Chat History** - Removes all messages but keeps the conversation
- **Delete Conversation** - Completely removes the conversation

## 💡 Pro Tips

1. **Real-time Updates**: Messages appear instantly without refreshing!
2. **File Attachments**: Support for PDF, DOC, DOCX, JPG, PNG (up to 3 files)
3. **Mobile Friendly**: Works great on phones and tablets
4. **Keyboard Shortcuts**: Press Enter to send messages
5. **Status Tracking**: Watch checkmarks to know when messages are read

## 🐛 Troubleshooting

### Messages not sending?
- Check your internet connection
- Ensure you're logged in
- Refresh the page and try again

### Can't see conversations?
- Make sure you've started a chat from a project
- Check that the project owner (client) exists

### Chat button not working?
- Ensure you're logged in as a company/freelancer
- Check that the backend server is running
